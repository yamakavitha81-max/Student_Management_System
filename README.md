# Student Management System

## About the Project

This is a Student Management System developed using Python. I created this project to practice Python concepts such as functions, loops, lists, dictionaries, file handling, and JSON.

The program allows users to store and manage student records through a menu-driven interface. Student data is saved in a JSON file so it is available even after the program is closed.

## Features

* Add a new student
* View all students
* Search a student using Student ID
* Update student details
* Delete a student record
* View student statistics (highest, lowest, average marks)
* Sort students by name or marks
* Display the topper student
* Input validation for age, marks, and duplicate IDs

## Technologies Used

* Python
* JSON
* File Handling

## Files

* `student_management_system.py` – Main program
* `students.json` – Stores student data
* `README.md` – Project description

## How to Run

Run the following command:

```bash
python student_management_system.py
```
